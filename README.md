# engx — Nexus Developer Platform

Local-first developer runtime — start, observe, and trust your system.

## Install

```bash
curl -fsSL https://get.engx.dev/install.sh | bash
```

## Upgrade

```bash
engx upgrade
```

## Quick start

```bash
engxd &                    # start the platform daemon
engx status                # check all services
engx doctor                # diagnose any issues
```

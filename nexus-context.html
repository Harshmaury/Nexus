ADD ANY NEW PROJECT IN 3 MINUTES
Nexus is language-agnostic. These steps work for Go, Python, .NET, Node, Rust, Java — any stack.
1
Create project folder
$mkdir -p ~/dev/projects/my-new-project && cd ~/dev/projects/my-new-project && git init && git remote add origin https://github.com/Harshmaury/my-new-project.git
2
Add project manifest (.nexus.yaml)
Create this file in your project root. Nexus reads it to understand the project.
# .nexus.yaml — place in project root name: my-new-project type: web-api # web-api | microservice | ai | tool | library language: go # go | dotnet | python | node | rust version: 1.0.0 paths: root: ~/dev/projects/my-new-project src: src/ tests: tests/ commands: start: go run ./cmd/server/ build: go build ./... test: go test ./... lint: golangci-lint run ports: - 8090: api dependencies: runtime: [postgres, redis] # what must be running keywords: # for Nexus Drop detection - "package main" - "my-new-project"
3
Register with Nexus
$engx register ~/dev/projects/my-new-project
Nexus reads .nexus.yaml, adds to SQLite registry, starts watching for health, enables Drop routing. Done.
4
Add to context dump
Open context-dump.sh and add one line to the PROJECTS array:
